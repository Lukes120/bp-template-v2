## Come catturare gli screenshot

1. Apri l'app in browser massimizzato (1920x1080 raccomandato)
2. Vai sulla schermata target
3. Usa **Win+Shift+S** (Windows Snipping Tool) per ritagliare l'area pertinente
4. Salva direttamente in questa cartella con il nome esatto della tabella sopra

## Cosa succede se mancano

La guida funziona lo stesso: al posto dell'immagine appare un placeholder grigio
con il nome del file mancante, così sai cosa aggiungere.

## Formato

- **PNG** preferibile (qualità migliore per UI screenshot, lossless)
- Dimensioni: nessun vincolo, l'immagine si adatta in larghezza alla guida
- Risoluzione: meglio non sotto i 1200px di larghezza per leggibilità su monitor grandi
